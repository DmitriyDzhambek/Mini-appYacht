# Папка для логирования действий пользователей

Эта папка содержит систему логирования всех действий пользователей в приложении "Прощай лень".

## Структура

- `logger.js` - Основной файл с системой логирования
- `actions.json` - Файл с сохраненными действиями пользователей (создается автоматически)
- `README.md` - Этот файл с документацией

## Функциональность

### Основные возможности:
- Автоматическое сохранение всех действий пользователей
- Логирование кликов, изменений в полях ввода, ошибок
- Сохранение в localStorage для персистентности
- Экспорт логов в JSON формат
- Статистика действий пользователей
- Фильтрация по действиям и датам

### Логируемые события:
- `app_loaded` - Загрузка приложения
- `app_unloaded` - Выход из приложения
- `button_click` - Клик по кнопке
- `input_changed` - Изменение в поле ввода
- `javascript_error` - Ошибки JavaScript
- `screen_changed` - Переключение экранов
- `snack_added` - Добавление перекуса
- `diary_entry_added` - Добавление записи в дневник
- `bike_status_updated` - Обновление состояния велосипеда
- И другие пользовательские действия

## Использование

```javascript
// Логирование действия
saveUserAction('custom_action', {
    data: 'some_data',
    value: 123
});

// Получение всех логов
const logs = window.userLogger.getAllLogs();

// Получение статистики
const stats = window.userLogger.getStats();

// Экспорт логов
window.userLogger.exportLogs();
```

## Хранение данных

- Логи сохраняются в localStorage под ключом `proshaylen_user_logs`
- Максимальное количество логов: 1000
- Старые логи автоматически удаляются при превышении лимита
- ID сессии сохраняется в sessionStorage

## Безопасность

- Чувствительные данные не логируются
- Пароли и личная информация фильтруются
- Все логи хранятся локально в браузере пользователя
