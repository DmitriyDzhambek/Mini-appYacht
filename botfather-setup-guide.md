# 🤖 Настройка VeloPath Mini App в BotFather

## 📋 Пошаговая инструкция

### Шаг 1: Настройка GitHub Pages

1. **Перейдите на GitHub:**
   ```
   https://github.com/DmitriyDzhambek/Mini-appYacht
   ```

2. **Настройки Pages:**
   - Нажмите **Settings** (вверху)
   - В левом меню выберите **Pages**
   - **Source:** Deploy from a branch
   - **Branch:** main
   - **Folder:** /root
   - Нажмите **Save**

3. **Ожидание (1-2 минуты):**
   - GitHub развернет проект
   - Появится зеленая галочка
   - URL будет доступен

### Шаг 2: Проверка развертывания

**Откройте URL:**
```
https://dmitriydzhambek.github.io/Mini-appYacht
```

**Должно работать:**
- ✅ Загружается страница
- ✅ Виден морской дизайн
- ✅ Анимированные волны
- ✅ Кнопка "Прощай лень"

### Шаг 3: Настройка BotFather

1. **Откройте BotFather в Telegram:**
   ```
   @BotFather
   ```

2. **Выберите вашего бота:**
   - `/mybots`
   - Выберите **MiniYacht_Bot**

3. **Настройте WebApp:**
   - `/setmenubutton`
   - Выберите **MiniYacht_Bot**
   - **Button text:** `VeloPath`
   - **Web App URL:** `https://dmitriydzhambek.github.io/Mini-appYacht`

4. **Или настройте через /newapp:**
   - `/newapp`
   - Следуйте инструкциям
   - **Short Name:** `velopath`
   - **Title:** `VeloPath`
   - **Description:** `Зарабатывай награды за активность`
   - **URL:** `https://dmitriydzhambek.github.io/Mini-appYacht`

### Шаг 4: Тестирование

**Откройте бота:**
```
https://t.me/MiniYacht_Bot
```

**Проверьте:**
- ✅ Кнопка меню "VeloPath"
- ✅ Mini App открывается
- ✅ Дизайн работает
- ✅ Анимации видны
- ✅ Звуки работают
