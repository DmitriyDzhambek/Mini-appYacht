# 🚀 Создание нового Vercel проекта - VeloPath Mini App

## 📋 Инструкции для развертывания

### 1. Удаление старых проектов (ручная)
1. Перейдите на https://vercel.com/dmitriydzhambeks-projects
2. Удалите все старые проекты вручную
3. Очистите dashboard

### 2. Создание нового проекта
1. Перейдите на https://vercel.com/new?teamSlug=dmitriydzhambeks-projects
2. **Import Git Repository**
3. Выберите: **GitHub** → **DmitriyDzhambek/Mini-appYacht**
4. **Project Settings:**
   - **Name:** `velopath-mini-app`
   - **Framework Preset:** `Other`
   - **Root Directory:** `/`
   - **Build Command:** (оставить пустым)
   - **Output Directory:** (оставить пустым)
   - **Install Command:** (оставить пустым)

### 3. Environment Variables (если нужны)
```
NODE_ENV=production
```

### 4. Нажмите **Deploy**
